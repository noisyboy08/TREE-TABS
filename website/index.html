Sowser

A canvas-style browser workspace built with WPF and WebView2. This site is hosted on the web; the app itself runs only on your PC after you download it.

Download

Replace the link below with your GitHub Releases asset URL (zip of published build) or Microsoft Store link. Vercel cannot run .exe files for visitors — they download and install locally.

Download for Windows Source / releases

Requirements

Windows 10 or 11, 64-bit. WebView2 Runtime (usually already installed with Edge). .NET 8 if you distribute framework-dependent builds.

Deploy this folder to Vercel: create a project, set the root directory to website, framework “Other”, no build command. Publish the desktop app separately (e.g. dotnet publish -c Release then zip the output) and attach it to GitHub Releases.
